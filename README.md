# macOS Security Home Lab

## Objective
Create a beginner-friendly macOS security hardening lab and document security controls using screenshots as evidence.

## System
- Device: macOS laptop
- OS: macOS (Ventura / Sonoma)

## Security Controls Implemented

### System Updates
- Automatic macOS updates enabled

### Device Protection
- Firewall enabled
- FileVault disk encryption enabled
- Secure screen lock configured

### Account Security
- Strong password policy
- Two-Factor Authentication (2FA) enabled on GitHub

### Browser Security
- Tracking protection enabled
- Unnecessary extensions removed

## Evidence
Screenshots are stored in the `/evidence` folder.

## Reflection
This lab documents real security controls commonly required in enterprise environments.
